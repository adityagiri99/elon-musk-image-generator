# Elon Musk Image Generator 🤖

This project converts any random image into an Elon Musk-style image using pixel brightness mapping.

## Features
- Upload any image
- Converts it into Elon Musk image
- Simple GUI using Tkinter

## Technologies Used
- Python
- Tkinter
- PIL (Pillow)
- NumPy

## How to Run
1. Install dependencies:
   pip install pillow numpy

2. Run the program:
   python elon.py

## Output
A new window will show the generated Elon Musk image.
